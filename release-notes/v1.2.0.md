# WorldShot Log v1.2.0 Release Notes

WorldShot Log v1.2.0 をリリースしました。

## 変更点

- 画像メタデータに含まれるタイトル情報を、`プリントのノート` として取り込めるようにしました。
- カードの詳細モーダルに、`プリント` ラベルと `プリントのノート` 表示を追加しました。
- 検索バーを改修し、`World` / `メモ` / `プリントのノート` の切り替え検索に対応しました。
- 検索バーに `検索` ボタンと `クリア` ボタンを追加し、Enter キーでの検索確定にも対応しました。
- 検索入力中にカード側へフォーカスが吸われないように調整しました。
- `ワールド一覧` の切り替えボタンをサイドメニュー上部へ移動しました。
- 設定のメンテナンスに、年月を指定して既存画像の情報を再取り込みできる機能を追加しました。
- サムネイル再生成と再取り込みの月ドロップダウン、ボタン幅、重なり順を調整しました。
