# WorldShot Log

> この README は、現時点の機能と仕様を整理するための開発用メモ寄りの内容です。公開向けの体裁は後で別途整える前提です。

WorldShot Log は、VRChat で撮影したスクリーンショットを整理・閲覧・補助管理するための Windows 向けデスクトップアプリです。  
Electron をベースにしており、月ごとの一覧表示、ワールド情報の取得、ラベル管理、背景画像設定などを 1 つのアプリで扱えます。

## 現在できること

### 画像の取り込み

- 画像 / フォルダのドラッグ＆ドロップ取り込み
- 設定した更新対象フォルダを `更新` ボタンで再走査
- 同一画像の重複判定
- 元画像パスの移動検出と復旧

### 一覧表示

- 年 / 月単位での写真一覧表示
- 年を押すと年単位表示、月を押すと月単位表示
- 日付ごとのグループ表示
- ライトモード / ダークモード切り替え
- フォント切り替え
- 背景画像の設定

### フィルター / 操作

- お気に入りフィルター
- 向きフィルター
- ラベルフィルター
- World 名のテキスト検索
- 複数選択モード
- Shift 範囲選択
- ドラッグ選択
- 一覧画面でのキーボード移動
  - 矢印キーで移動
  - Enter でカードモーダルを開く

### カードモーダル

- 拡大表示
- 前後画像への移動
- VRChat ページを開く
- 元画像を開く
- 保存先フォルダを開く
- お気に入り切り替え
- ラベル設定
- メモ保存
- ワールド名 / World URL の手動編集
- World 情報の再読み込み

### World 情報

- 取り込み後にバックグラウンドで World 情報を自動取得
- 同じ `worldId` の写真ではキャッシュを再利用
- 取得済み情報の自動反映
- 説明文、タグ、World URL の表示

### 設定 / メンテナンス

- 更新対象フォルダの追加 / 削除
- サムネイル再生成
- 表示中の月を削除
- サムネイルキャッシュを削除
- 全登録を削除
- DB 初期化
- アプリ内アンインストール導線
  - アンインストール
  - データも削除してアンインストール

## 現在の仕様

### World 情報の扱い

- 自動取得対象は public / labs 系の World 情報です
- private / non-public world は取得対象外です
- World 情報が取得できない場合は、一覧とモーダルで `ワールド名を取得できませんでした` を表示します

### 自動アップデート

- GitHub Releases を使った更新確認に対応しています
- Windows の配布版アプリでのみ動作します
- 開発実行 (`npm start`) では動作しません

### インストール / アンインストール

- 現在の配布形式は Squirrel.Windows ベースです
- インストール先の手動選択には未対応です
- アプリ内からアンインストール開始ができます

## データ保存場所

現在の保存先は以下です。

- DB / 設定  
  `app.getPath('userData')\\data\\`
- サムネイル  
  `%USERPROFILE%\\WorldShot Log\\thumbnails`

Windows 環境では概ね以下になります。

- DB / 設定  
  `C:\\Users\\<ユーザー名>\\AppData\\Roaming\\WorldShot Log\\data\\`
- サムネイル  
  `C:\\Users\\<ユーザー名>\\WorldShot Log\\thumbnails`

## 開発環境

- Electron 41
- better-sqlite3
- ExifReader
